Obook

Desktop App

Obook is a project in partnership with EDRLab linked with the Readium project. The Readium project was started by the IDPF in 2012 in order to create an open-source implementation of an EPUB 3.0 parser and rendering toolkit. Initially written in Javascript as a Google Chrome extension. It soon evolved with the addition of a C++based native SDK, for use in iOS, Android and desktop applications.

The aim of the project was to design a responsive reading app for desktop at first. It was a short project for a week. For this project, I made the main wireframes of the application (in blue), and some final designed screens, as well as a logo and a graphic chart.
